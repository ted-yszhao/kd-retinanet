\chapter*{Abstract}
I investigate whether knowledge distillation can make a compact RetinaNet detector more effective without making it more expensive to deploy. This problem matters because RetinaNet models with \texttt{ResNet-50} or \texttt{ResNet-101} feature pyramid backbones usually learn stronger representations than lighter alternatives, but that extra capacity also increases memory use and inference cost. My focus is on whether a frozen teacher detector can pass useful multi-scale structure to a smaller \texttt{ResNet-18}-based student while keeping the deployed model lightweight.

I train the student with the standard RetinaNet detection objective together with a feature distillation term applied to aligned FPN representations. Instead of matching raw backbone activations, I distill after the FPN, where teacher and student features already play the same semantic role and share the same channel width. This makes it possible to compare selected pyramid levels directly without inserting an extra learned adapter. To transfer structure rather than only activation magnitude, I use an SSIM-based loss on normalized, channel-aggregated feature maps so that the student is encouraged to reproduce local spatial organization that is relevant to dense object detection.

The final system includes configuration-driven training, checkpointing, recorded evaluation metrics, and COCO-style validation. Under a resource-constrained schedule of $10$ epochs with per-epoch subset sampling, a vanilla \texttt{ResNet-18} RetinaNet reaches $0.1515$ bounding-box mAP on the COCO validation set, while a \texttt{ResNet-101} teacher distilled into the same \texttt{ResNet-18} student reaches $0.1636$ mAP without increasing checkpoint size. The distilled student also improves AP$_{50}$, AP$_{75}$, and the small, medium, and large object metrics. However, a \texttt{ResNet-50} teacher distilled into the same \texttt{ResNet-18} student collapses to zero AP, showing that this kind of structural feature distillation is sensitive to teacher-student pairing and optimization stability. I therefore conclude that post-FPN SSIM-based distillation is a promising but not automatically robust way to improve compact RetinaNet detectors.
