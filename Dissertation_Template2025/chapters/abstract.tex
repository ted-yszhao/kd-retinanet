\chapter*{Abstract}
This dissertation investigates whether knowledge distillation can make a compact RetinaNet detector more effective without making it more expensive to deploy \cite{hinton2015distilling,lin2017focal}. This problem matters because RetinaNet models with \texttt{ResNet-50} or \texttt{ResNet-101} feature pyramid backbones usually learn stronger representations than lighter alternatives, but that extra capacity also increases memory use and inference cost \cite{he2016resnet,lin2017fpn,lin2017focal}. The central question is whether a frozen teacher detector can pass useful multi-scale structure to a smaller \texttt{ResNet-18}-based student while keeping the deployed model lightweight.

\ArchitectureFigure{Compact summary of the teacher-student RetinaNet distillation architecture highlighted in this abstract.}{fig:abstract-architecture}

The student is trained with the standard RetinaNet detection objective together with a feature distillation term applied to aligned FPN representations \cite{lin2017fpn,lin2017focal}. Instead of matching raw backbone activations, distillation is performed after the FPN, where teacher and student features already play the same semantic role and share the same channel width. This makes it possible to compare selected pyramid levels directly without inserting an extra learned adapter. To transfer structure rather than only activation magnitude, an SSIM-based loss is applied to normalized, channel-aggregated feature maps so that the student is encouraged to reproduce local spatial organization that is relevant to dense object detection \cite{wang2004ssim,derijk2022structural}.

The final system includes configuration-driven training, checkpointing, recorded evaluation metrics, and COCO-style validation \cite{lin2014coco}. Under a resource-constrained schedule of $10$ epochs with per-epoch subset sampling, a vanilla \texttt{ResNet-18} RetinaNet reaches $0.1515$ bounding-box mAP on the COCO validation set, while a \texttt{ResNet-101} teacher distilled into the same \texttt{ResNet-18} student reaches $0.1636$ mAP without increasing checkpoint size. A student-only \texttt{ResNet-50} baseline reaches $0.1830$ mAP, and the corresponding \texttt{ResNet-101} $\rightarrow$ \texttt{ResNet-50} distilled run improves that score further to $0.1954$. However, a \texttt{ResNet-50} teacher distilled into the same \texttt{ResNet-18} student collapses to zero AP, showing that this kind of structural feature distillation is promising but still sensitive to teacher-student pairing and optimization stability. The implementation, configurations, and recorded artefacts discussed throughout the dissertation are available in the public GitHub repository at \url{https://github.com/ted-yszhao/kd-retinanet}. Overall, post-FPN SSIM-based distillation appears to be a credible way to improve compact RetinaNet detectors, but not yet a universally robust one.
